A simple, educational keylogger project built using Python and the pynput library.
This tool is designed ONLY for learning purposes, cybersecurity awareness, and understanding how keylogging techniques work.

USE ONLY ON SYSTEMS YOU OWN OR HAVE PERMISSION TO TEST.

Features:
- Logs normal keys and special keys
- Timestamped logging
- Auto-stop after configurable minutes
- Saves logs into a file
- Creates backups when file grows large
- JSON configuration support
- Consent prompt included

Requirements:
pip install pynput

How to Run:
python main.py

Stopping:
- Press ESC
- Auto-stop by timer
- Close terminal
